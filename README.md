## 介紹

**DB Playground** 是一個線上資料庫語法練習平台，你可以在這裡練習 SQL 語法，並且查看執行結果。**DB Playground** 對於輸入的指令長度無限制且具備人性化貼心設計。支援 MySQL、PostgreSQL 等多種資料庫。
